# :desktop_computer: Patch GUI :desktop_computer:
Simple UI System for ElectrosmithDaisy Patch with a scrollable menu.

It looks like this:
```
----------------------------
| Header 1        Header 2 |
|--------------------------|
| > MenuItem1      value   |
|   MenuItem2      value   |
|   MenuItem3      value   |
|   MenuItem4      value   |
|   MenuItem5      value   |
----------------------------
```
See [patchGui.h](patchGui.h) for more documentation.
